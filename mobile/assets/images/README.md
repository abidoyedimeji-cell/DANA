# App assets

Add the following files for builds and splash:

- **icon.png** – 1024×1024 app icon
- **splash-icon.png** – splash screen image (e.g. logo)
- **adaptive-icon.png** – 1024×1024 Android adaptive icon foreground
- **favicon.png** – web favicon

You can copy and resize from the web app’s `public/` folder (e.g. `icon.svg`, `apple-icon.png`).
